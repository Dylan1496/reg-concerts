# Welcome to the Reg concert repository!

There are two categories in this area:

- reg-sets, covering any show on the Elton John Salvation server done by Reg
- non-reg-sets, covering any show on the Elton John Salvation server done by a person other than Reg

Shows are listed chronologically.

## Format of pages

Each show has a page dedicated to it. They will have the whole setlist, as well as any notes that the person that performed wishes to provide. An embed to the video on YouTube will also be provided (if available).
